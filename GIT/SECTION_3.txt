1. Create a new branch 'js-assignments'.
ANS. git branch js-assignments

2. Switch to 'js-assignments'
ANS. git checkout js-assignments

3. Copy all in Assignments inside 'Assignments' folder.
ANS. git checkout master -- JS_Assignments

4. Commit JS assignments into 'js-assignments' branch.
ANS. git add . && git commit -m "JS_Assignments  submitted"

5. Make changes in files.
ANS. Changes done.

6. Commit those changed files.
ANS. git add . && git commit -m "JS_Assignments changed"

7. Switch to master branch.
ANS. git checkout master

8. Make changes into README.txt file and commit into master.
ANS. Changes done.
     git add . && git commit -m "README file is changed"

9. Again switch to 'js-assignments' branch.
ANS. git checkout js-assignments

10. Make changes to files.
ANS. Changes done

11. Commit those changes.
ANS. git add . && git commit -m "Final JS_Assignments is submitted"

12. Switch to master.
ANS. git checkout master

13. Merge 'js-assignments' to master.
ANS. git merge js-assignments

14. Finally delete 'js-assignments' branch.
ANS. git branch -d js-assignments
